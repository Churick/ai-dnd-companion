# AI D&D Companion

Бесплатное приложение для игры в Dungeons & Dragons с искусственным интеллектом в роли Мастера.

## Возможности (план)
*   **AI Dungeon Master:** Локальная языковая модель (через Ollama) ведет сюжет и отвечает на действия игрока.
*   **Генерация изображений:** Стабильная диффузия (локально или через Replicate) визуализирует ключевые сцены.
*   **Управление состоянием:** Система отслеживает персонажа, инвентарь и контекст мира.
*   **Веб-интерфейс:** Удобный чат-интерфейс для игры из браузера.

## Технологический стек (предварительный)
*   **Фронтенд:** React + Vite + Tailwind CSS
*   **Бэкенд:** Python FastAPI
*   **База данных:** SQLite (разработка), PostgreSQL (продакшен)
*   **AI Текст:** Ollama (Llama 3 / Mistral)
*   **AI Изображения:** Replicate API (Stable Diffusion) / Локальный ComfyUI
*   **Развертывание:** Docker, Vercel (фронтенд), Railway (бэкенд)
